AI-Resume-Insight 🚀

⚙️ Environment Variables

Create a .env file in the root directory.

GOOGLE_API_KEY=YOUR_GOOGLE_GEMINI_API_KEY
OPENAI_API_KEY=YOUR_OPENAI_API_KEY

Example:

GOOGLE_API_KEY=AIzaSyxxxxxxxxxxxxxxxx
OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxxxx

⚠️ Never commit your .env file to GitHub.

Add this to .gitignore:

.env


cd AI-Resume-Insight

npm install

Backend:

pip install -r requirements.txt

Run:

python app.py
🎯 Project Goal

AI-Resume-Insight aims to help students, freshers, and professionals create ATS-friendly resumes, improve technical skills through coding assessments, and receive AI-driven career guidance to increase employability and interview success.
